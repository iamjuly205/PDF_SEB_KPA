Nội dung đề thi trống.

Các câu hỏi sau khi trích xuất hoặc thêm thủ công sẽ xuất hiện tại đây theo định dạng chuẩn A4.

Cấu hình Google Gemini API

Để trích xuất đề thi từ ảnh chụp chất lượng cao, ứng dụng sử dụng AI từ mô hình Gemini 2.5 Flash của Google.

Làm thế nào để lấy API Key miễn phí?

  1. Truy cập vào Google AI Studio và đăng nhập bằng tài khoản Google.
  2. Click chọn nút **"Get API key"** màu xanh dương ở góc trên bên trái.
  3. Chọn **"Create API key"**, chọn dự án hoặc tạo dự án mới và sao chép mã Key (bắt đầu bằng `AIzaSy...`).
  4. Dán mã đó vào ô nhập liệu ở trên và bấm **Lưu cấu hình**.
Bảo mật: API Key của bạn chỉ được lưu cục bộ trên trình duyệt này và không bao giờ được gửi tới bất kỳ máy chủ trung gian nào ngoại trừ máy chủ API chính thức của Google.
